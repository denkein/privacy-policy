1. Какие данные могут собираться

Приложения могут обрабатывать техническую информацию, необходимую для их работы, улучшения качества сервиса и обработки встроенных покупок.

  • технические данные устройства и системы;
  • диагностическая информация об ошибках и сбоях;
  • данные, связанные со встроенными покупками, предоставляемые магазином приложений;
  • внутренние данные приложения, связанные с прогрессом пользователя, статистикой, серией дней, достижениями и настройками.

2. Для чего используются данные

  • для корректной работы приложений;
  • для сохранения пользовательского прогресса и статистики;
  • для обработки встроенных покупок и восстановления доступа к платным функциям;
  • для исправления ошибок и улучшения стабильности приложений.

3. Передача данных третьим лицам

Часть данных может обрабатываться сторонними платформами, если это необходимо для работы приложений, встроенных покупок, аналитики, публикации или технической диагностики. К таким платформам могут относиться магазины приложений и их платёжные сервисы.

4. Хранение данных

Пользовательский прогресс, настройки и часть служебной информации могут храниться локально на устройстве пользователя. Данные, связанные со встроенными покупками, могут дополнительно обрабатываться магазином приложений в рамках его платёжной инфраструктуры.

5. Права пользователя

Пользователь может запросить информацию об обработке своих данных, а также запросить удаление данных, если это применимо.

По вопросам, связанным с персональными данными, можно написать разработчику по адресу: denkein@gmail.com

6. Удаление данных

Для удаления локальных данных приложения пользователь может удалить приложение с устройства или очистить его данные через настройки устройства. Если требуется дополнительный запрос по данным, пользователь может связаться с разработчиком по указанному email.

7. Изменения политики

Разработчик может обновлять настоящую политику конфиденциальности. Актуальная версия всегда публикуется на этой странице.
